# Autonomous Vehicle Sim

Simulation-based project for an **autonomous vehicle** (self-driving car): sensing the environment and driving with little or no human input. This repository is named **`autonomous-vehicle-sim`** to reflect its focus on simulator workflows and autonomous driving behavior.

# STEPS to run this project:


## STEP 01:
Clone the repository

```bash
git clone https://github.com/YOUR_GITHUB_USERNAME/autonomous-vehicle-sim.git
cd autonomous-vehicle-sim
```

*(Replace `YOUR_GITHUB_USERNAME` with your GitHub username or organization.)*

## STEP 02:
Create an environment & activate


```bash
conda create -n autonomous-vehicle-sim python=3.7 -y
conda activate autonomous-vehicle-sim
```

## STEP 03:
Install the requirements


```bash
pip install -r requirements.txt
```


## STEP 04:
Start the drive with the following command


```bash
python drive.py
```

Open the simulator in **Autonomous** mode and run when prompted.
